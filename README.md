### 목표 기능

1. 인디음악 앨범 정보 제공, 검색 기능 - maniaDB api 사용
2. 앨범 분류 : 카테고리(장르) - 최신순, 인기순 (1일, 1주, 1달 기준)
3. 앨범에 대한 여러 타사이트의 리뷰보기/혹은 리뷰로 가는 링크
4. 음악 취향이 비슷한 유저와 매칭 대화 기능 - 채팅 기능
5. 앨범이나 곡 유튜브로 리다이렉트 / 유튜브 플레이어 api 사용
6. 닉네임 자동 추천

<br>

1. Database

   - 검색 최적화
2. 서버

3. 기능
   - 외부 API 사용, 외부 정보 크롤링
   - 실시간 인기 검색어 🤞

   - 비속어 감지 및 등록 방지
   - 채팅 기능 🤞
